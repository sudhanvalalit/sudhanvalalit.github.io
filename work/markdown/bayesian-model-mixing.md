# Bayesian Model Mixing

![Bayesian Model Mixing](../img/Computation/comp-bg.jpeg)

Combining predictions from multiple theoretical models is often more reliable than trusting any single model, especially when each is only valid in a limited regime. My work develops open-source software for Bayesian model mixing and combination, letting theoretical predictions transition smoothly between models while propagating uncertainty.

## Key Publications

### pybmc: Bayesian Model Combination

- A lightweight Python package for combining predictions from multiple models using Bayesian model averaging and mixing techniques.
- [GitHub Repository (pybmc)](https://github.com/sudhanvalalit/pybmc)
